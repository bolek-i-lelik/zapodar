Контроллеры:

1. AdminController - контроллер главной страницы админки
	1.1 behaviors - правила доступа
	1.2 Index - главная страница админки
	(Остальное пока лучше не трогать, возможно будут изменения)

2. ArticleController - контроллер вывода статей 
	2.1 behaviors - правила доступа
	2.2 Index - страница с перечнем всех статей
	2.3 View - просмотр отдельной статьи
	2.4 Create - добавление новой статьи
	2.5 Update - редактирование статьи
	2.6. Delete - удаление статьи

3. DostupController - контроллер разграничения "админ - не админ"

4. NewsController - Контроллер вывода новостей
	4.1 behaviors - правила доступа
	4.2 Index - страница с перечнем всех новостей
	4.3 View - просмотр отдельной новости
	4.4 Create - добавление новой новости
	4.5 Update - редактирование новости
	4.6. Delete - удаление новости

5. ProductsController - контроллер вывода продуктов
 	5.1 behaviors - правила доступа
	5.2 Index - страница с перечнем всех продуктов
	5.3 View - просмотр отдельного продукта
	5.4 Create - добавление нового продукта
	5.5 Update - редактирование продукта
	5.6. Delete - удаление продукта

6. SiteController - здесь всё понятно.

7. UserController - контроллер просмотра юзеров
	7.1 behaviors - правила доступа
	7.2 Index - страница с перечнем всех юзеров
	7.3 View - просмотр отдельного юзера
	7.4 Create - добавление нового юзера
	7.5 Update - редактирование юзера
	7.6. Delete - удаление юзера
